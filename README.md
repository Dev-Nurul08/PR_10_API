# Weather App

A responsive weather application that provides real-time weather information for cities around the world using the OpenWeatherMap API.

## Features

- **Real-time Weather Data**: Get current weather conditions including temperature, humidity, and wind speed
- **City Search**: Search for weather information by entering any city name
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Visual Weather Icons**: Displays intuitive weather icons for better user experience
- **Detailed Weather Information**: Shows feels-like temperature, humidity percentage, and wind speed

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid layouts)
- JavaScript (ES6+)
- OpenWeatherMap API
